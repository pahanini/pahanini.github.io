Как изменить File Modification Date/Time у PDF файла

2025-08-07T13:37:24+05:00 | 1 минута чтения | Обновлено 2025-08-07T12:38:23+05:00

@ pahanini

Понадобилось изменить дату создания pdf файла. Не достаточно просто сделать touch.

touch -a -m -t 202001071200.00 example.pdf

Эта изменит лишь даты доступа и изменения файла. Помимо этого даты модификации хранятся в exif заголовках PDF.

Для просмотра и изменения exif метаданных можно воспользоваться exiftool .

Просмотр метаданных exiftool example.pdf

exiftool example.pdf 
ExifTool Version Number         : 13.30
File Name                       : example.pdf 
Directory                       : .
File Size                       : 200 kB
File Modification Date/Time     : 2025:08:07 09:54:58+05:00
File Access Date/Time           : 2025:08:07 09:54:58+05:00
File Inode Change Date/Time     : 2025:08:07 09:54:57+05:00
File Permissions                : -rw-r--r--
File Type                       : PDF
File Type Extension             : pdf

Меняем дату exiftool -FileModifyDate="2020:01:07 12:00:00" example.pdf.

Результат:

exiftool example.pdf 
ExifTool Version Number         : 13.30
File Name                       : example.pdf 
Directory                       : .
File Size                       : 200 kB
File Modification Date/Time     : 2020:01:07 12:00:00+05:00
File Access Date/Time           : 2025:08:07 09:54:58+05:00
File Inode Change Date/Time     : 2025:08:07 09:54:57+05:00
File Permissions                : -rw-r--r--
File Type                       : PDF
File Type Extension             : pdf
